Display Sample Data in a Heat Map

Gene expression data, RBM data, and any other data that can be displayed in a Dataset Explorer heat map can also be displayed in a Sample Explorer heat map.  You can select one or more datasets for display.

To display sample data in a heat map:

1.  If the dataset of interest is not included in the result set, refine the search by selecting additional search filters (see Select and Remove Search Filters).

2.  When a dataset of interest appears, click the dataset’s row.

Note:  Do not click the dataset name.  Clicking the dataset name will open Dataset Explorer, and you will lose any search filters you selected in Sample Explorer.

3.  To display data from multiple datasets, use these techniques:

□    To select non-sequential rows of datasets, press and hold down the Ctrl key, then click each row to include.

□    To select sequential rows of datasets, press and hold down the Ctrl key to select the top- or bottom-most row, then hold down the Shift key and click the row at the opposite end.

4.  To deselect a selected row, hold down the Ctrl key and click the row again.

5.  When all rows to include are selected, click Advanced Workflow > Heatmap.

A dialog box like the following appears:

6.  Select the parameters for the data to be displayed, or keep the defaults.

For information about the parameters, see Requirements for Generating Heat Maps.

Note:  You can only select parameters for Subset 1.

7.  Click Run Workflow.

The visualization may require a minute or more to process, depending on the amount of data to be displayed.  The processing time is shown at the bottom-right of the window:

8.  Click OK when the completion dialog box appears.

Again, it may take some time for the visualization to be downloaded and displayed.
