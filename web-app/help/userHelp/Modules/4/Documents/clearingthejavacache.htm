Clearing the Java Cache

Clearing your Java cache will force the most recent versions of tranSMART components, including heat maps and other GenePattern visualizations, to be retrieved and displayed.

To clear the Java cache:

1.  Open the Windows Control Panel using Start > Control Panel (Start > Settings > Control Panel if you are using the Classic Start menu).

2.  Double-click Java.

The Java Control Panel appears.

3.  Click Settings under Temporary Internet Files.

4.  Click Delete Files.

The following dialog is displayed:

5.  Select the type of files you wish to clear, then click OK.
