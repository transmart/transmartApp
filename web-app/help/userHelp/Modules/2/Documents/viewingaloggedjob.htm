Viewing a Logged Job

Each advanced analysis that you have run in the previous seven days will be logged in the Jobs tab. You may view the visualization again by selecting it from the list.

To run a logged advanced workflow:

1.  Run tranSMART, then click the Dataset Explorer tab.

2.  In the right pane, click the Jobs tab:
