Disabling the Login Prompt for Trusted Sites

After upgrading to IE8 for tranSMART, you may find that you are being prompted to log into JnJ sites that did not previously prompt you to enter login credentials.

To disable the login prompt for trusted sites in IE8:

1.  Log into the site by typing your login ID and password.

2.  In the IE8 status bar, double-click the security settings button (one of the following):

 

The Internet Security Properties dialog appears.

3.  Click Trusted Sites, then click Sites.

The Trusted Sites dialog appears.

4.  If the URL for the site you are on does not appear in the Websites box, click Add to add it:

5.  Click Close.

6.  Click Custom Level on the Internet Security Properties dialog.

The Security Settings – Trusted Sites Zone dialog appears.

7.  Scroll to the bottom of the list and click User Authentication > Login.

8.  Select Automatic logon with current user name and password.

9.  Click OK.

10. Click OK on the Internet Security Properties dialog.
