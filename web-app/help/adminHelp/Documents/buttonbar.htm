Button Bar

The button bar above the right pane of the Faceted Search page lets you display different views of the data and manage the contents of the Search Results view.

The figure below shows all the possible buttons on the Faceted Search page. Note that Collapse All and Options appear only in Search Results view:

Button

Description

Home

Displays summary charts showing the number of studies and analyses in the data warehouse for a selected data category. Also shows lists of saved filters and cross-trial analyses.

The summary charts are not affected by the search query defined in the Active Filters area. However, the number of analyses are affected by the Show only significant results option that you set from the Options button.

For information, see Overview of the Faceted Search UI.

Search Results

Displays search results based on the filters defined in the Active Filters area.

For information, see Viewing Search Results.

Cross Trial Analysis

Displays data from multiple analyses that you select in Search Results view.

For information, see Viewing a Cross-Trial Analysis.

(0)

Displays the number of analyses currently selected for a cross-trial analysis, or 0 if no analyses are selected.

Button also lets you deselect some or all selected analyses.

Collapse All

Collapses any open studies and analyses in Search Results view.

Button is visible only in Search Results view.

Options

Lets you make the following global settings:

■   The maximum number of probe sets or genes and proteins that can be displayed in a visualization of an analysis.

This global setting can be overridden for a particular analysis.

■   Specify whether result sets contain only statistically significant results.

Button is visible only in Search Results view.
