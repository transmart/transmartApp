Browse the Analysis List

Only those analyses that satisfy the search query defined in Active Filters can be listed for a study. You can view the entire list of analyses using the scroll bar on the page.

You can perform the following tasks for an analysis:

■    View metadata for the analysis.

To do so, click the icon to the left of the analysis name:

■    View the threshold for statistically significant data.

The threshold is specified in the QA Criteria property of the analysis metadata:

 

If the Show only significant results option is selected from the Options button on the button bar, the only records that will be displayed for an analysis are those whose probe set results meet or exceed the threshold.

■    Select an analysis for inclusion in a cross-trial analysis.

To do so, select the check box to the left of the analysis name:

■    Open an analysis.

When you open an analysis, you can view analysis data in a variety of visualizations and export the data to a comma-separated text file.

To open an analysis, click the analysis name.
