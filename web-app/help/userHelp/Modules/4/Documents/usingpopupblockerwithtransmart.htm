Using Pop-Up Blocker with tranSMART

You can use all tranSMART features, including heat maps and other GenePattern visualizations, while your browser has pop-up blocker enabled.

To use tranSMART with your browser’s pop-up blocker enabled:

1.  In Internet Explorer 8, click the Tools menu, then click Internet Options.

2.  Click the Privacy tab.

If you have pop-up blocker enabled, Turn on Pop-up Blocker is selected in the Pop-up Blocker group box.

3.  In the Pop-up Blocker group box, click Settings.

4.  In Address of website to allow, type the following:

   jnj.com

 

5.  Click Add.

6.  Click Close, then click OK to close the Internet Options dialog.

Note:  If your security software blocks pop-ups, refer to the security software’s documentation to learn how to allow pop-ups from jnj.com.
