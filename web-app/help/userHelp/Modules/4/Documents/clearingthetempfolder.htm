Clearing the Temp Folder

Removing temporary files helps save disk space. Each file in the Temp folder has been classified as no longer needed by Windows. All files are safe to delete.

To remove files from the Temp folder:

1.  Click Start > All Programs > Accessories > Run.

2.  Type %temp% into the prompt, then click OK.

The Temp folder appears.

3.  Do one of the following:

□    To select all items, hold down the Ctrl + A keys together.

□    To select specific items, use the standard Windows selection techniques (click/Ctrl-and-click, click/Shift-and-click).

4.  Click Delete, then click Yes to confirm.

Note: If you are prompted that there are hidden files in the folder, click OK to dismiss.

5.  Navigate to your desktop and right-click the Recycle Bin, then click Empty Recycle Bin.
