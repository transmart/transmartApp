The Workspace

The workspace section of dataset explorer allows user to save comparison criteria and analysis elements.

The workspace screen is broken into two sections.

Subset Manager

The Subset Manager section contains a list of saved sets of comparison criteria. Details for what exactly has been saved can be found by hovering over the icon in the 'Query' column.

There are three ways to use a set of saved comparison criteria

1. Click the radio button in the 'Use' column. This will automatically populate the subsets in the comparison tab with the saved criteria.

2. Click the mail icon in the 'Email' column. This will allow the user to mail a link to the comparison criteria to the address of their choosing. When the link is clicked, the user will be brought to tranSMART with the comparison tab open and the subsets pre-populated

3. Click the link icon in the 'Link' column. This allows the user to copy/paste the link mentioned in #2 above.

Report/Analysis Manager

The Report/Analysis Manager works similarly to the Subset manager, but contains saved analysis parameters instead of comparison criteria.

The 'Type' column tells the user what type of analysis was saved, while the 'Query' column allows the user to expose the parameter details before loading the analysis.

Clicking the icon in the 'Run' column will take the user to an analysis page with all parameters pre-populated. At this point the analysis will not have actually been run. The user can choose to edit the analysis or simply click run.
