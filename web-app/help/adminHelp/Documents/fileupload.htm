File Upload

Files can be uploaded to tranSMART by using the upload data wizard. Files will always be uploaded at the study level. Within the upload data wizard there are two options for uploading a file.

1. Upload file to faceted search

2. Upload file to dataset explorer

The workflow is identical between the two options with the exception of the 'Study' field providing options that correspond to the chosen selection

File upload is fairly straight forward. Users select a study be either typing into the autocomplete text box, or using the 'Browse' option to pick through a list of options. Users then click the 'Choose File' option to browse to the file on their local machine. Lastly, a File name and description need to be specified. These fields govern how the file will be viewed within the tranSMART UI

There are no restrictions on what types of files can be uploaded to the system.
