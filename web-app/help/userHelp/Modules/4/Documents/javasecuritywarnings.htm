Java Security Warnings

The following security warnings (see the links below) may appear when you attempt to generate a Dataset Explorer visualization such as a heat map or principal component analysis.  The warnings apply to software from the Broad Institute web site.  The Broad Institute, based at Harvard and MIT, is the developer of the GenePattern heat map engine, which Dataset Explorer leverages to display certain visualizations.

You can find information about the Broad Institute at the following site:

http://www.broadinstitute.org
