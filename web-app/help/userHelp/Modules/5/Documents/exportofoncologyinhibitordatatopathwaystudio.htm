Export of Oncology Inhibitor Data to Pathway Studio

In a tranSMART Literature search result, which contains oncology and asthma data curated by Jubilant, you can now export oncology inhibitor data to Pathway Studio, along with alterations and interactions data for both oncology and asthma.

To do so, click the Export ResNet button in a Literature search result that contains oncology inhibitor data.
