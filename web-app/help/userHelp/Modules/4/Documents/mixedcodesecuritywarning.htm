Mixed Code Security Warning

You may encounter the following Java Mixed Code security warning when you generate a visualization in Dataset Explorer:

If you click Yes, the visualization will not be displayed.

You have two options for displaying the visualization:

■    Click No on the warning dialog.

The visualization will be displayed, and the warning dialog will not reappear if you generate other visualizations during the current tranSMART session.  However, the next time you run tranSMART and attempt to generate a visualization, the warning will reappear.

■    Disable the warning, as described below.

If you disable the warning, the warning will no longer appear when you generate a visualization in the current tranSMART session or in new sessions.  However, the warning also will not appear when other applications encounter a mixed code security condition.

To disable this warning:

1.  Open the Windows Control Panel using Start > Control Panel (Start > Settings > Control Panel if you are using the Classic Start menu).

2.  Double-click Java.

3.  Click the Advanced tab.

4.  Open Security > Mixed code (sandboxed vs. trusted) security verification.

5.  Select Enable – hide warning and run with protections, as shown below:

6.  Click OK.
