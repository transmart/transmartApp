Creating a Gene/RSID List

A more simple alternative to creating a gene signature is to create a Gene/RSID list. These lists contain only gene symbols and rsids, but do not require the user to walk through all the metadata screens in the gene signature workflow

There are three basic ways to create a gene/rsid list:

 

1.  Add the list of genes and rsids for the list to a tab delimited text file.

2.  Copy and Paste a list of gene symbols and rsids into the first text box in the "Enter List Manually" section

List should be comma or new line separated and the form will expand as needed

2.  Manually enter lists of genes or rsids

After creating the gene/rsid list it will be usable anywhere in the application that a gene signature can be used.
